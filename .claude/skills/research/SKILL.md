# Skill: research — 記事テーマ・キーワードリサーチ

## 概要
`en/` フォルダの既存記事と重複しない新しい記事テーマを1つ選定し、3言語分のタイトル・slug・キーワードをJSON形式で出力する。

---

## 手順

### Step 1: 既存記事の確認
`en/` フォルダのHTMLファイル一覧を取得し、カバー済みトピックを把握する。

```bash
ls en/*.html
```

### Step 2: テーマ選定
以下の候補から、**既存記事と重複しない**テーマを1つ選ぶ。
ロングテール・低競合キーワードを優先する。

**テーマ候補プール:**
- 残業（zangyō）のルールと仕組み
- 契約途中で辞めたらどうなるか
- 5S（整理・整頓・清掃・清潔・躾）の実際
- 健康診断について知っておくこと
- 会社の寮（dormitory）の使い方
- 工場の夜勤（夜間シフト）の実態
- 工場での怪我・労災（rōsai）の対処法
- 技能実習生と特定技能の違い
- 工場勤務で使える日本語の数字・単位
- 日本の祝日と工場の休日スケジュール

候補プールが尽きた場合は、ターゲット読者（外国人工場労働者）に役立つ新テーマを自ら提案する。

### Step 3: JSON出力
以下のフォーマットで出力する（コードブロックなし・JSONのみ）:

```json
{
  "topic": "日本語でテーマ説明",
  "primary_keyword": "英語メインキーワード（ロングテール）",
  "secondary_keywords": ["関連キーワード1", "関連キーワード2", "関連キーワード3"],
  "slug": "ハイフン区切り英小文字-slug",
  "badge_category": "バッジカテゴリ名",
  "title_en": "英語タイトル",
  "title_tl": "タガログ語タイトル",
  "title_vi": "ベトナム語タイトル"
}
```

## バッジカテゴリ一覧
`Culture` / `Rules` / `Safety` / `Communication` / `Pay & Benefits` / `Daily Life` / `Visa & Documents` / `Useful Japanese`

---

## 注意事項
- slugは既存ファイル名と被らないこと
- タイトルは3言語とも**翻訳ではなく自然な表現**で書く
- primary_keywordは読者が実際に検索しそうな英語フレーズにする
